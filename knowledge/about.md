# About Bohdan Hodzenko

Bohdan Hodzenko is a full-stack developer with a passion for building products that are as delightful to use as they are solid under the hood. He cares deeply about performance, accessibility, and clean code that teammates enjoy reading.

## Background

Bohdan has been building web applications professionally for several years, working across the full stack — from pixel-perfect interfaces to resilient back-end services. He has experience working on SaaS products, developer tools, and open-source projects.

**Backend:** Node.js, Go, REST APIs, GraphQL
**Databases:** PostgreSQL, Redis, Supabase
**Infrastructure:** Docker, Vercel, CI/CD pipelines
**Design:** Figma, design systems, component libraries
**Other:** Accessibility (WCAG), performance optimisation, open source

## Values

Bohdan prioritises:

- **Accessibility** — building for everyone, not just some users
- **Clean code** — readable, maintainable, well-tested
- **Developer experience** — tools and APIs that other developers enjoy using
